CRI-O

Mar 7, 2021
1 min read
container kubernetes

What

cri-o是一個container engine(frontend)，由Redhat領導

Why

CRI-O is an implementation of the Kubernetes CRI (Container Runtime Interface) to enable using OCI (Open Container Initiative) compatible runtimes. It is a light CRI-O 是實作 Kubernetes CRI的container engine，讓Kubernetes可以使用任何符合OCI介面的container runtime，目前主要支援

  1. kata container
  2. runc

但理論上，任何符合OCI的Container Runtime都可以支援。

簡單的說，CRI-O 是

  1. Redhat領導的
  2. Kubernetes優先的

一個CRI container engine實作

More About

CRI-O是由多個開源的軟體來組成的，包含

  1. OCI compatible runtime
  2. containers/storage
  3. containers/image
  4. networking (CNI)
  5. container monitoring (conmon)
